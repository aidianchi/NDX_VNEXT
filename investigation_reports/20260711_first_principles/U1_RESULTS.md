# U1 证据敏感性实验：完整结论（2026-07-11 ~ 07-12）

## 实验设计

对同一份真实快照（2026-07-10 collect，`output/data/data_collected_v9_live.json`）构造扰动副本，离线重跑完整分析管线（`--data-json --skip-report`，非 official），观察判断是否跟随证据。共 10 个 run：

| Run | Prompt | 输入 | 目的 |
| --- | --- | --- | --- |
| baseline ×3 | 旧 | 原始快照 | 基线 + 同输入随机漂移标尺 |
| P1 | 旧 | 估值反转（PE 35.9→22.5、分位 81→22） | 单轴敏感性 |
| P2 | 旧 | 广度反转（新低38、McClellan −38.5） | 单轴敏感性 |
| P3 | 旧 | 流动性反转（实际利率 2.31→0.3 等） | 单轴敏感性 |
| P4 | 旧 | P1+P3+广度健康+集中度正常+一致性修正 | 多轴组合 |
| P5 | 旧 | P4+信用/情绪良性（质量利差 99.9 分位→30） | 零看空轴定罪实验 |
| ab_baseline | 新 | 原始快照 | 验证不过度矫正 |
| ab_p5 | 新 | P5 快照 | 验证敢转多 |

扰动脚本与改动清单：本目录 `u1/`（`perturb_snapshot.py`、`build_p4.py`、`build_p5.py`、`change_log*.json`）。产物：`output/analysis/vnext/u1_*`。

## 核心结论

1. **层级分析是证据驱动的（强阳性）**。每个被扰动层的结论都如实转向，置信度合理变动（P3 中 L1 restrictive→expansionary 且 conf 升 high；P5 中 L2/L3 全面转正）。层级 prompt 无姿态锚定（状态词汇表对称），与此一致。
2. **旧 prompt 下最终动作层被结构性锁死在谨慎区**。三次单轴扰动均维持"谨慎/小比例试探"；P4 抓住未扰动的真实极端信号（CCC−BB 质量利差 99.87 分位走阔）转谨慎（合理）；P5 零看空轴下仍谨慎，理由是"盈利验证缺失"。机械成因：旧 prompt 规定"高赔率"须五类证据共同支持，而盈利预期数据永久缺失（Wind PIT 无返数）→ **建设性结论在结构上不可能出现**；叠加三份综合 prompt 的范文全部是恐慌市谨慎姿态（范例锚定强于指令服从）。
3. **prompt 重写后双达标（commit `2443be5`）**。新 prompt × P5：主导面首次为 opportunity、"赔率中性偏有利"、动作为持仓+对冲+适度加等权（建设性但不鲁莽，且诚实点名残留的估值分歧）。新 prompt × 真实快照：仍"赔率不利、防御为主"且更果断（降高贝塔、20-30% 现金），未过度矫正。桶级翻转远超基线×3 的自然漂移。赔率判断变为具名记分卡；矛盾 ID 摆脱范例锚定。
4. **DataIntegrity 闸门对数值篡改零反应**（10 个 run 全部 93.7% publishable）——闸门只测结构完整性不测数值语义，独立重算校验带（工单#3）的直接依据。
5. **claim gate 不稳定**：同输入基线三连的 verified 率 7/8、1/8、7/8；P5 也是 1/8。校验结果受 LLM 措辞随机性支配，已立工单。
6. **遗留半个病根**：新 prompt 解决了"说不出口"，但盈利预期数据缺失仍是真实缺口——利多世界里"等盈利确认"仍会是常驻反方。彻底解决靠证据菜单工单（Wind PIT 盈利预期取数，NEXT_STEPS P0）。

## 有效性边界

- 扰动数据非真实市场组合，仅检验"结论是否跟随证据"，不检验"结论是否正确"。
- 新旧对照各方向仅 1 个 run（桶级判定在基线方差之外，但细粒度措辞差异不可过度解读）。
- P1 扰动有已知内部矛盾（Danjuan 分位漏改），P2 混入方向不一致信号；均已在 P4/P5 修正。
- LLM 为 DeepSeek（生产同款）；不同模型的姿态锚定强度可能不同。
